ThermalDecoder — краткая инструкция для Windows
Версия: 0.0.3-beta

1. Распакуйте архив ZIP в любую папку (например «Рабочий стол» или «Загрузки»).

2. Откройте папку ThermalDecoder. Внутри будет файл ThermalDecoder.exe и другие служебные файлы — это нормально, их удалять не нужно.

3. Запустите программу двойным щелчком по ThermalDecoder.exe.

4. Python устанавливать не нужно — всё необходимое уже внутри.

Если антивирус или «Защитник Windows» предупреждает о неизвестном приложении: это типично для программ без подписи издателя. Решение о запуске принимаете вы; при необходимости добавьте исключение в настройках антивируса.
